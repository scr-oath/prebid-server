# prebid-server fork — engineering site

This site is a **fork-local** companion to [prebid/prebid-server](https://github.com/prebid/prebid-server).
It hosts performance reports and engineering writeups produced from this
fork that aren't ready to live in the upstream repo's `docs/` tree.

For canonical prebid-server documentation, see:

- [prebid/prebid-server](https://github.com/prebid/prebid-server) — main repo
- [docs.prebid.org](https://docs.prebid.org/prebid-server/overview/prebid-server-overview-2.html) — user docs

## What lives here

- **[Reports](./reports/)** — performance and architecture writeups,
  each linked to an upstream issue or RFC discussion.
